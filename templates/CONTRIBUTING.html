Contributing

Can't find a cheatsheet for your language or tool?

The creation and moderation of these cheatsheets is a community effort. Once you get up to speed on whatever subject you were looking for, consider contributing a cheatsheet to the learnxinyminutes-docs repository.

Having an issue with the extension itself?

Submit an issue on GitHub, or if you find an issue and know the solution, feel free to fork this repo and submit a pull request. The same goes for feature requests!
